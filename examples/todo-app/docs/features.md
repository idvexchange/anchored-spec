# To-Do App — Feature Specification

## Overview

A client-side to-do list application that lets users manage tasks through add, complete, delete, and filter operations.

## Features

| ID | Feature | Priority |
|----|---------|----------|
| REQ-1 | Add new tasks with a text input | must |
| REQ-2 | Display all tasks as a list | must |
| REQ-3 | Toggle task completion on/off | must |
| REQ-4 | Delete individual tasks | must |
| REQ-5 | Filter tasks by status (all/active/completed) | should |
