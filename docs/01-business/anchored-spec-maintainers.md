# Anchored Spec Maintainers

This team owns the Anchored Spec framework and the reference repository patterns used to demonstrate it.

## Responsibilities

- keep the framework generic across repositories and programming languages
- maintain the CLI, runtime, and reference modeling guidance
- keep the repository harness examples thin, explicit, and repo-local
- align docs, tests, and scaffold behavior when public workflow guidance changes
