# Contributing to Anchored Spec

Thank you for your interest in contributing! See the full [Contributing Guide](docs/contributing.md) for development workflow, project structure, code conventions, and testing.

## Quick Start

```bash
git clone https://github.com/idvexchange/anchored-spec.git
cd anchored-spec
pnpm install
pnpm verify    # Build + test + lint
```
